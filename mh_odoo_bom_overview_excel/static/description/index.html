MaplesHub BoM Overview Excel Export

Overview

Export the exact data from Odoo’s BoM Overview to Excel — including parent and sub-BoMs and all components — with one click.

Highlights

  • Same values as the official BoM Overview (no custom recomputation)
  • Single flat Excel with parents, sub-BoMs, and component lines
  • Columns: Product, Country Codes, Qty (UoM), Lead Time (d), Route, BoM Cost, Product Cost

How to Use

  1. Open a Bill of Materials in Odoo
  2. Click the “Export BoM Overview (XLSX)” button (or call the route directly)
